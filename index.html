This wiki was made as a gift to the pbf community.

This wiki contains most information about Project Bronze Forever (pbf) but since this is a solo project, some information might be incorrect or missing so if you found any, you can report it through my email darealducke@gmail.com

Pokemon Brick Bronze (pbb) was a roblox pokemon-like game created by tbradm, lando64000, zombie7737, KyleAllenMusic, Shipool, kevincatssing, Roselius, Srybon, Our_Hero, chrissuper, MySixthSense, roball1 and oldschooldude2, the game was released in October 24, 2015 but got banned and discontinued due to copyright by Nintendo in April 18, 2018.

Project Bronze Forever is a copy version of Pokemon Brick Bronze which are being developed by toasty, faithful, mg35t with tanoshi being the main server owner and some other more.
